# Taiwan Blue Energy Reality Atlas

This repository contains the public distribution build of an interactive 3D atlas exploring Taiwan's energy landscape, offshore wind development, fisheries coexistence, ports, subsea cables, marine ecology, and energy history.

## Live Experience

Open the judge-ready website at:

https://pinsyuan1019.github.io/taiwan-blue-energy-atlas/

## Highlights

- Interactive 3D terrain and satellite map of Taiwan
- Energy-layer exploration across multiple generation types
- Historical energy timeline with visual state changes
- Four location-based Changhua coastal scenarios
- Field hotspots for turbines, vessels, fisheries, ports, cables, and ecology
- Built-in energy guide for common questions
